# Paper Poster 1:2:1

用于设计和组织 1:2:1 学术海报：左窄、中宽、右窄，适合方法型研究故事。

## 它能做什么

- 把论文、摘要、提纲或结果整理成 1:2:1 海报方案。
- 在正式写作前先建立“论文内容 -> 海报内容”的对应矩阵。
- 建立基于原文的文本 ledger，避免海报信息量过少。
- 先生成 3 个以上配色方案和预览图，再等待用户选择。
- 结合公开可检索的论文/海报配对样本来压缩内容。

## 适用场景

当你需要这些内容时，使用这个 skill：

- 1:2:1 学术海报，
- betterposter 风格海报，
- conference poster 方案，
- 论文到海报的抽象流程，
- 基于 corpus 的海报写作流程。

## 核心流程

1. 先提炼出论文的一个核心主张。
2. 建立对应矩阵：哪些内容要放大、压缩、重排或省略。
3. 从原文中抽取足量重要文字：动机、方法、实验、结果解释和 caveat。
4. 锁定 1:2:1 结构。
5. 生成 3 个以上配色方案，并等待用户选择。
6. 应用 corpus 抽象规则。
7. 先写海报计划，再进入正式绘制。
